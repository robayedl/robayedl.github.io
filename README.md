# Robayed Ashraf — Portfolio

Personal portfolio of **Robayed Ashraf**, AI/ML Engineer based in Sydney, Australia. Showcases projects in Computer Vision, Agentic AI, and LLM Applications.

Live at [robayedl.github.io](https://robayedl.github.io)

## Stack

- **Vite + React 18** — fast build tooling, component-based UI
- **Tailwind CSS** — utility-first styling with a custom dark design system
- **Framer Motion** — scroll-driven entrance animations and micro-interactions
- **HTML Canvas** — per-project generative tech animations (neural, wave, grid, scan, etc.)
- **GitHub Actions** — automatic deployment to GitHub Pages on push to `main`
- **Claude Code** — used along the way to speed up the build

## Features

- Animated project cards with demo videos (Google Drive + YouTube embeds)
- Scroll-linked timeline for experience
- Responsive — mobile-first layout with device-adaptive particle background
- PDF resume viewer built into the nav

## Development

```bash
npm install
npm run dev      # http://localhost:5173
```
